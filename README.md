zjuthesis
=========

Latex template for zju thesis

具体使用方法请阅读 [thesis.pdf](https://github.com/ibillxia/zjuthesis/blob/master/thesis.pdf) 文档。

BugFix：
在使用之前需要对ctex-xecjk-winfonts.def 文件进行修改，这里你只需要将本项目中的 ctex-xecjk-winfonts.def 文件拷贝到以下目录（覆盖之前请自行备份）：

`%CTEXDIR%/MiKTeX/tex/latex/ctex/fontset`

其中%CTEXDIR%是CTeX的安装目录。
